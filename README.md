# Real State Frontend

Frontend application for the Real State lead management system.

## Live App

https://react-state-frontend.vercel.app/

## Tech Stack

- React
- Vite
- Redux Toolkit
- React Router

## Environment Variables

Create a .env file in frontend root:

VITE_BASE_URL=https://real-state-backend-tegd.onrender.com/api/v1

## Run Locally

1. Install dependencies:

npm install

2. Start development server:

npm run dev

App runs on http://localhost:5173 by default.

## Build

npm run build
